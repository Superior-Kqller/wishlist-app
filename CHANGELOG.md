# Changelog

Все значимые изменения документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект использует [Semantic Versioning](https://semver.org/lang/ru/).

## [1.2.0] — 2026-03-18

### Добавлено

- **Glassmorphism UI** — стеклянные эффекты для карточек, хедера, диалогов
- **Mesh gradient** — улучшенный фон страницы с радиальными градиентами в dark mode
- **Новые анимации** — shimmer, glow-pulse, float, slide-up keyframes
- **A11y: prefers-reduced-motion** — отключение анимаций для чувствительных пользователей
- **A11y: aria-live** — озвучивание изменений списка для screen readers

### Изменено

- Header: стеклянный эффект с backdrop-blur-xl
- WishlistCard: glass-card стиль с hover glow, semantic --success color
- Dialog: усиленный overlay (60% opacity)
- Typography: text-wrap balance для заголовков
- Font: добавлены fallback шрифты для Manrope

### Рефакторинг

- `colorThemes` вынесены в `src/lib/themes.ts`
- PriorityStars: span вместо disabled button для read-only режима (a11y)

## [1.1.2] — 2026-03-18

### Изменено

- UI: убрана кнопка «Добавить из ссылки» в хедере, так как парсер маркетплейсов признан нестабильным и не является частью официального функционала

## [1.1.1] — 2026-03-18

### Исправлено

- **Загрузка аватара (локально)** — кнопка «Сохранить» теперь корректно активируется после выбора файла
- **Обновление аватара без кеша** — `avatarUrl` сохраняется с cache-busting параметром (`?v=...`), чтобы новое изображение отображалось сразу после загрузки

## [1.1.0] — 2026-03-17

### Изменено

- **Права видимости товаров** — товары без подборки (`listId = null`) больше не отображаются в общей выдаче. Товар виден только если привязан к подборке, доступной пользователю
- `GET /api/items` исключает `listId: null` из фильтрации
- `GET /api/items/:id` проверяет доступ через `canUserSeeItem`
- `GET /api/items/export` экспортирует только товары из видимых подборок
- `canUserSeeItem()` теперь возвращает `false` для товаров без подборки
- UI: предупреждение при выборе «Без подборки» в форме добавления товара

### Добавлено

- Unit-тесты для `list-utils.ts` (10 тестов: canUserSeeList, canUserSeeItem, getVisibleListIdsForUser)

## [1.0.1] — 2026-03-17

### Исправлено

- **Парсер Ozon** — переход на внутренний API (`composer-api.bx`) вместо HTML-скрапинга. Устраняет ошибку «цепочка перенаправлений» при парсинге товаров Ozon
- Поддержка коротких ссылок Ozon (`/t/...`) — автоматическое разворачивание
- Приоритет цены: карта Ozon → обычная → без скидки
- HTML fallback при недоступности API

## [1.0.0] — 2026-03-17

Первый релиз.

### Основные возможности

- **Аутентификация** — NextAuth.js с JWT-сессиями, ролями USER/ADMIN
- **Вишлист** — CRUD для желаний с изображениями, ценами, приоритетами (1-5 звёзд), тегами, заметками
- **Подборки** — создание списков с шарингом между пользователями
- **Парсер URL** — автоматическое извлечение данных товаров с Wildberries (API), Ozon, AliExpress, generic-сайтов
- **Серверный поиск** — ILIKE по названию, заметкам и тегам с debounce 300ms
- **Бесконечная прокрутка** — cursor-пагинация + Intersection Observer
- **Фильтры в URL** — search, sort, tags, purchased синхронизируются с query params
- **Быстрый приоритет** — hover-превью и клик по звёздам прямо на карточке
- **Bulk-операции** — режим выделения, массовое удаление и отметка «куплено»
- **Экспорт** — CSV и JSON через /api/items/export
- **Темы** — 7 цветовых тем + светлый/тёмный/системный режим
- **Адаптивный дизайн** — мобильные фильтры, touch-свайп галереи
- **Администрирование** — управление пользователями, сброс паролей
- **Настройки профиля** — аватар, имя, смена пароля
- **Комментарии** — к каждому желанию
- **Статистика** — дашборд с ценами по валютам

### Безопасность

- SSRF-защита парсера — DNS resolve + проверка приватных IP-диапазонов
- Rate limiting через Valkey (Unix socket) с in-memory fallback
- CSP nonce для inline-скриптов
- Bcrypt (12 rounds), Zod-валидация, защита от удаления последнего админа

### Инфраструктура

- Docker multi-stage build, GitHub Actions → GHCR
- PostgreSQL 17 + Prisma 7 (driver adapters)
- Valkey 8.1 для rate limiting (Unix socket)
- Автоматический seed пользователей при первом запуске
